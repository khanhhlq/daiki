### 🍄 Ngôn ngữ lập trình và công nghệ

- Dự án được phát khai và xây dựng trên 3 ngôn ngữ lập trình:

  - HTML 👨‍💻
  - CSS 🎨
  - Javascript 🧉

- Những công nghệ được áp dụng:
  - API Iframe Youtube 📫
  - Bootstrap 💨
  - Fontawesome 🍦
  - Embed Spotify 🔮
  - Embed Discord 🌐
